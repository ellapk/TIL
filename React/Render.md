# **Render 함수**
- 리액트에서 UI를 사용할 때 필연적으로 눈에 보이는 화면을 구성해줌.

- 컴포넌트의 정보를 이용해 화면을 구성(렌더링)한다.

## **Rendering 순서**
1. 컴포넌트 정의
2. 컴포넌트의 정보를 이용해 렌더링

3. 문자열 형식의 html 코드 반환

4. DOM 요소에 주입





## **React가 Rendering을 실행하는 과정**

1. props가 변경되었을 때

2. state가 변경되었을 때

3. forceUpdate()를 실행하였을 때

4. 부모 컴포넌트가 렌더링되었을 때
